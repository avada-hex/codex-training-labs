# Frontend

This folder contains the React patient experience for Capstone Team 5.

## Structure

- `src/components` for shared UI elements
- `src/pages` for route-level screens
- `src/routes` for protected routing
- `src/services` for API calls
- `src/types` for shared frontend types
- `src/utils` for small reusable helpers

## Frontend Focus

- Authentication pages
- Protected patient views
- Patient records
- Vitals and symptoms updates
- Suggested care plans
- Dashboard views
- Contact us form
